void parse ( char *  line)

Função parse do programa.

Inicia-se um apontador de caracteres (delimitadores).

Posteriormente, através de um ciclo, inicia-se uma nova variável (um apontador de carateres (token)), com a condição de paragem quando esse token for NULL. A cada iteração o token é atualizado.

Através de condições if verifica-se o valor a devolver ao topo da pilha, se à partida, a string contiver apenas números é imediatamente devolvida, caso contrário, através da função 'strcmp' verificar-se-à que valor/es terão que se devolver.

Returns
Não devolve nada (void).

void PRINT_STACK ( )

Função PRINT_STACK do programa.

Inicia-se um ciclo for, onde uma variável é incializada também, a condição de paragem ocorre se a variável é maior que o topo da da Stack, a cada iteração incrementa-se a variável.

É imprimido o valor da Stack no índice da variável, de acordo com o seu valor a cada iteração.

Não devolve nada (void).
