Função parse do programa. 

Inicia-\/se um apontador de caracteres (delimitadores).

Posteriormente, através de um ciclo, inicia-\/se uma nova variável (um apontador de carateres (token)), com a condição de paragem quando esse token for N\+U\+LL. A cada iteração o token é atualizado.

Através de condições if verifica-\/se o valor a devolver ao topo da pilha, se à partida, a string contiver apenas números é imediatamente devolvida, caso contrário, através da função \textquotesingle{}strcmp\textquotesingle{} verificar-\/se-\/à que valor/es terão que se devolver.
